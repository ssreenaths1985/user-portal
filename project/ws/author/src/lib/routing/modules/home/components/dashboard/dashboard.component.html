Authoring Tool

Welcome, you can use this tool to create new contents or modify existing ones.

  • Use 'Create' to start creating a new content. Once the content is saved, it would be available in Draft.
  • The contents sent for review, would be available in 'Sent for Review' folder. To edit the content sent for review, it needs to be recalled.
  • The contents approved by the Reviewer would be available in 'Publish' folder under 'Action Items' section.
  • Published contents would be available in 'Published' section. Once you start editing, the content will be available in the Draft section.
